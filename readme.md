## 1. Instalación

* npm i bootstrap@5.3.2
* npm install -g sass

## 2. Compilar
Esto es solo para mantener actualizado el style.css por lo que cada vez
que se modifique, se debe compilar.
* sass --watch scss/_variables.scss:scss/styles.css
